vk bot
=======
бот для VK на Node.js



зависимости
-----
* vksdk
* lodash
* request
* q
* zombie


запустить
-------
node app


управлять 
-------
команды:

`list` список всех комманд

`auth` обновить token

`listen` начать/прекратить смотреть сообщения


авторизация
--------
написать всё в `auth.json`
как в примере `auth.json.example`
и выполнить команду `auth`


начать общение
--------
начинает слушать автоматом
n команда `listen`
